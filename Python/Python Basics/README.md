# Python Basics

This folder contains the lessons, labs and small projects I completed during the Python Basics section on FreeCodeCamp.

## Topics Covered

- Variables and Python naming conventions
- Strings, integers, floats and Boolean values
- Lists, tuples, sets and dictionaries
- Mutable and immutable data types
- Type checking with `type()` and `isinstance()`
- String indexing, slicing and common string methods
- Arithmetic and comparison operators
- Functions, parameters and return values
- Local and global scope
- Conditional statements using `if`, `elif` and `else`
- Boolean operators using `and`, `or` and `not`
- Truthy and falsy values
- Input validation and unit testing

## Labs and Projects

- Bill Splitter
- Build an RPG Character
- Caesar Cipher
- Discount Calculator
- Employee Profile Generator
- Movie Ticket Booking Calculator
- Report Card Printer
- Travel Weather Planner

## What I Learned

This section helped me understand the fundamentals of Python and how to combine variables, functions, calculations, strings and conditional logic to build small programs.

I also learned the importance of validating function arguments, returning values instead of only printing them, and matching the exact output expected by automated unit tests.

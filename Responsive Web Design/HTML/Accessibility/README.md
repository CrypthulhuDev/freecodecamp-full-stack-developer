# 2026-07-01 - HTML Accessibility

## Overview

This section focused on improving accessibility in HTML by learning how to make web pages easier to use for all users, including people who rely on screen readers, keyboard navigation, captions, labels, and clear page structure.

The main focus was on general accessibility principles, ARIA attributes, WCAG guidelines, semantic HTML, readable content, and building pages that are easier to navigate and understand.

* Clear headings and page structure
* Descriptive links and buttons
* Debugging accessibility issues
* Improving user experience for different users and devices
